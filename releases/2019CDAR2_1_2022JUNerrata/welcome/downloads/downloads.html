What is in a V3 Download?

The ballot package includes several documents. To facilitate faster downloading, the ballot package has been broken into several smaller downloads. To ensure that most of the hyperlinks within the ballot documents work, there is a Core download (see below) package that includes all the reference and help documents including:

  • Package Note to Readers
  • Version 3 Guide
  • Refinement, Constraint and Localization
  • Datatypes Abstract Specification
  • Glossary
  • Reference Information Model
  • Vocabulary Domains document
  • HL7 Development Framework
  • Infrastructure Domains
  • ITS XML and UML, CTS, GELLO and CDA
  • Supporting Graphics, Javascript and Cascading Stylesheets (for display)

Hyperlinks in the ballot content to documents that you've not downloaded will not work. Thus, all reviewers are encouraged to download the Full Site package or at a minimum the Core download and the appropriate domains.

Advice on V3 Downloads

The V3 Ballot Packages are comprised of several files and documents that hyperlink to one another. To ensure that the hyperlinks work, we recommend that you download the full package. If you decide not to download the entire package, be advised that hyperlinks to documents that you've not downloaded will not work. This package contains thousands of files, making the process time consuming based on the connection speed you have to the internet. A high speed line will yield about a 20 minute (estimated) download. Once downloaded, the installation only takes about another 5-10 mintues.

The Core files download provides all the Foundation Documents for the ballot (plus a few additional for convenience since not all documents available are being balloted) and all the support files needed in order for domain downloads to work.

When downloading any of the available packages, for convenience, you may wish to create a directory for the downloads, although this is not required. Also, be sure to allow for quite a bit of space on your hard drive, especially for the larger packages. The Full package extracts to more than 500 Megabytes.

Advice on V3 Installation

Once any of the packages have been downloaded from the HL7 web site, you must use a .zip decompression program to open the .zip file to retrieve its contents.

When you unzip the files you download, you should create a central "root" directory on your hard drive to which you will then extract the zip file contents. Once established, you should only use that directory for future installations or patches. If you do not, there is always the risk that hyperlinks connecting the documents will not work. We recommend setting up a directory named based on the ballot you are reviewing, for example, "C:\HL7V3BALLOT_2010SEP".

Once completed, find the sub-directory where you installed the files. You will see an HTML directory if that is the version of the package you downloaded. In this directory will be an INDEX.HTM file. This is your launching point to the site. Please note that the issues link always links to the main ballot site on www.hl7.org. This is because the issues page is a 'dynamic' page that is open to ongoing changes throughout the ballot period.
